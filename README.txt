Fuel Finder Growth Machine - Full UI Version

Purpose:
This is a safe growth helper for launching Fuel Finder as a free product.
It helps you find Facebook group keywords, score groups, generate copy-ready posts,
create UTM links, and track posting history.

Files:
- growth_machine_full.html

How to use:
1. Open growth_machine_full.html in Chrome/Safari.
2. Use Group Keyword Finder to generate group search terms.
3. Click Open FB Search to manually search groups.
4. Score groups with member count/activity/fit.
5. Generate post copy and copy it.
6. Manually paste into Facebook groups.
7. Save posting log.
8. Export CSV if needed.

Important:
- This tool does not scrape Facebook.
- This tool does not auto-post or send DMs.
- This is intentional to avoid account bans and spam behavior.

Recommended daily rule:
- Post only 2-3 groups per day.
- Change wording slightly per group.
- Reply to comments manually.
